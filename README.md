## Telmedicine

# For backend

Go to the backend directory ---
```
cd backend

```
then 

```
npm i

```

# To start the server of backend

```
npm start
```

# For Frontend

Go to frontend Directory...

```
cd frontend
```
then

```
npm i --force
```
to start the server of frontend

```
npm start
```
